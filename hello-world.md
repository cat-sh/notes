# The history of "Hello, World!"

- Dates back to 1972, to Brian Kernighan at Bell Laboratories

- Kernighan used the phrase in a document called "A Tutorial Introduction to the Language B"

```
main( ) {
    extrn a, b, c;
    putchar(a); putchar(b); putchar(c); putchar('!*n');
}
 
a 'hell';
b 'o, w';
c 'orld';
```

- Gained popularity in 1978 with the publication of "The C Programming Language"

```
#include <stdio.h>

main()
{
	printf("hello, world!\n");
}
```

- Acts as a useful example with which to compare languages

- It can teach a lot about the syntax of a language

- Can help confirm the compiler/interpreter is running properly on the user's system
